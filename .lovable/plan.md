## Design System
- **Background**: Off-white/cream `#F5F0EB`, with CSS grain texture
- **Text**: Dark charcoal `#1a1a1a`
- **Accent**: Warm beige/tan for highlights
- **Headings**: Playfair Display, uppercase, very large (5-8rem for hero)
- **Body**: Inter, clean and light
- **Images**: Rounded organic containers with soft shadows
- **Animations**: Scroll-triggered fade-ins, floating image parallax, smooth section transitions

## Technical
- Single-page with smooth scroll navigation
- Responsive (mobile-first)
- Intersection Observer for scroll animations
- Google Fonts (Playfair Display + Inter)
